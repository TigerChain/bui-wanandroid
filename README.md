## 课程简介
本套视频教是 bui 的进阶课程，我们使用 bui 框架来开发一个玩 android app ，采用 wanandroid 给我们暴漏出来的真实 api 去开发，并且本套课程采用视频+笔记的形式呈现给大家，方便大家查阅
